# V27a 路线记录：单拍通用 LUT Q7 MUL

## 目标

验证“单拍通用 Q7 乘法”是否能在不使用 DSP 的情况下达到 300 MHz。

## 实现方式

- `OP_MUL` 仍是普通 MCU 乘法指令。
- 使用 `ex_abs_b[7:0]` 控制 8 个移位部分积。
- 通过组合加法树求和，再根据符号右移 7 位得到 Q7 结果。
- 没有新增 FFT/蝶形/复数/91 专用指令或模块。

## 验证

| 项目 | 结果 |
| --- | ---: |
| 官方样例 + 20 随机 | PASS |
| `cnt_test` | 157 |
| WNS/TNS | -2.199 ns / -88.361 ns |
| LUT/FF/DSP | 1203 / 648 / 0 |

## 结论

功能正确但 300 MHz 不通过。该结果说明在当前 FPGA 和 no-DSP 约束下，把通用 Q7 乘法完全压成单拍会形成过长组合路径。
