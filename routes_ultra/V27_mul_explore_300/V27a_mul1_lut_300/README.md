# V27a_mul1_lut_300

V27a 将通用 Q7 `OP_MUL` 改成单拍 LUT 移位加法树，不使用 DSP，不新增专用指令。

## 修改内容

- 在 `rtl/mcu_core.v` 内生成 8 组部分积。
- 使用组合加法树完成 25-bit 数据乘 8-bit 系数。
- `OP_MUL` 在执行级直接写回。
- 切断 MUL 的 EX 前递，遇到紧邻 MUL 结果依赖时由 RAW hazard 插入气泡。
- 保持 `SYNTH_MAX_DSP=0`，Vivado 报告 DSP=0。

## 结果

| 项目 | 结果 |
| --- | ---: |
| 回归 | 官方样例 + 20 组随机 PASS |
| `cnt_test` | 157 |
| MCU 频率 | 300 MHz |
| 推算耗时 | 0.523 us |
| WNS | -2.199 ns |
| TNS | -88.361 ns |
| WHS | +0.093 ns |
| THS | 0.000 ns |
| LUT | 1203 |
| FF | 648 |
| DSP | 0 |

## 结论

V27a 的周期数很漂亮，但 300 MHz 时序明显失败，最差路径是通用乘法组合树写回路径，逻辑级数约 17 级。该路线只作为负结果保留，不建议上板或作为主线。
