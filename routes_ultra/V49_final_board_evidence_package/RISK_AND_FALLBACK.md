# 风险与兜底方案

## 方案一：最快已上板路线

使用 V45。

优点：当前最快，`cnt_test=85`，300 MHz timing clean，DSP=0，已经完成 no-ILA 下载和 ILA 板上验证。
风险：V45 是最新路线，调试材料较新，答辩时要明确区分 no-ILA 正式成绩和 ILA 调试证据。

## 方案二：稳定回退路线

使用 V42/V34。

优点：上板证据早已固化，`cnt_test=88`，300 MHz timing clean，DSP=0，和 V45 速度差距只有 3 个 cycle。
风险：不是当前最快路线，如果老师只看速度榜，V42 应作为回退而不是第一展示。

## 方案三：32-bit 合规展示路线

使用 V36 或 V49 中的合规说明材料。

优点：更适合老师重点检查机器码、指令宽度和 MCU 微架构位置时解释。
风险：V36 速度慢于 V45/V42，主要用于合规兜底，不用于速度榜第一展示。

## 方案四：最低风险已上板路线

使用 V22b。

优点：早期已上板路线，结构相对简单，风险最低。
风险：`cnt_test=173`，速度明显低于 V45/V42。

## 当前推荐

答辩主线：V45 作为实物展示和最终速度成绩，V42/V34 作为快速回退，V36/V49 合规材料用于回答 32-bit 指令和普通 MCU 架构问题。
