# 风险与兜底方案

## 方案一：最快已上板路线

使用 V42/V34。

优点：已经上板，ILA 证据完整，`cnt_test=88`，300 MHz timing clean，DSP=0。

风险：不是理论最快路线，V45 的 `cnt_test=85` 更快但尚未上板。

## 方案二：32-bit 合规展示路线

使用 V36 或 V49 中的合规说明材料。

优点：更适合老师重点检查机器码、指令宽度和 MCU 微架构位置时解释。

风险：V36 速度慢于 V42/V45，主要用于合规兜底，不用于速度榜第一展示。

## 方案三：最低风险已上板路线

使用 V22b。

优点：早期已上板路线，结构简单，风险最低。

风险：`cnt_test=173`，速度明显低于 V42/V45。

## 当前推荐

答辩主线：V42/V34 作为实物展示和最终成绩，V45 作为后续优化成果说明。

如果老师要求“必须全部以已上板为准”，只展示 V42/V34。

如果老师允许展示未上板优化潜力，可以补充 V45，但必须明确它尚未完成板上验证。
